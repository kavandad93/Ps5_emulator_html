# Ps5_emulator_html
An html file with folder, emulate the ps home, with _controller full support_

**specifications:**
- Html file
- 16:9 full support
- Full controller support
  - Buttons
  - Left joystick
  - R1/L1
- Create accounts
- Save accounts in .json file
- Install games from .Game File



***Notes:***
1. This file DO NOT CONNECT TO PSN
2. This file do not needs Internet
